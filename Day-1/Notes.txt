                             BOOTSTRAP
    ============================================================================

    =>Bootstrap is a component library(a pre written code for a specific purpose).
    => Bootstrap provides responsiveness.
    => offical website :- https://getbootstrap.com/

    => How to start /setup bootstrap :-
    1) goto official website --> get started --> copy cdn links & paste in your html head.

    link:- <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

    2) check wheather bootstrap working or not by giving class names btn-primary.

    colors of bootstra:
    -----------------------
    => Bootstrap uses 8 colors:-
    primary(blue), secondary(grey), success(green), danger(red), warning(yellow), info(skyblue), light(white), dark(black).

    => You can also customise on your own color.

    => Bootstrap works by grid layout of 12 columns 
    each line in class "row"
    each col can be made by "col" or "col-number".. max value is 12.
    => everything in bootstra in row.
